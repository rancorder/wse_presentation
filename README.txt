WSE ローカル台本操作型プレゼン

使い方:
1. index.html を Chrome / Edge で開く
2. 「顧客表示を開く」を押す
3. 開いた別ウィンドウだけを画面共有する
4. 元の画面でカンペをクリック、または次へ/前へで進行する

構成:
- assets/slide01.png
- assets/slide02.mp4  ← 動画
- assets/slide02.png  ← 動画ポスター/代替画像
- assets/slide03.png
- assets/slide04.png
- assets/slide05.png
- assets/slide06.png
- assets/slide07.mp4  ← 動画
- assets/slide07.png  ← 動画ポスター/代替画像
- assets/slide08.png
- assets/slide09.png

差し替え:
最終版の画像・動画ができたら、同じファイル名で assets フォルダに上書きしてください。

推奨:
画像: 1920x1080 PNG
動画: MP4 / H.264 / 音声なし / 5〜8秒 / 1920x1080
